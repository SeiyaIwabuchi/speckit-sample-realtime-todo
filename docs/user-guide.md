# Todo管理アプリケーション ユーザーガイド

## 概要

Todo管理アプリケーションは、リアルタイムで同期されるタスク管理ツールです。複数のデバイス間で即座に変更が反映され、タグによる分類やフィルタリング機能を提供します。

## 主な機能

- **リアルタイム同期**: 複数のデバイス間で即座に変更が反映されます
- **タグ管理**: Todoに複数のタグを付けて分類できます
- **フィルタリング**: タグによる絞り込み検索が可能です
- **認証**: メールアドレスとパスワード、またはGoogleアカウントでログイン
- **レスポンシブデザイン**: スマートフォンやタブレットにも対応

## 使い方

### 1. アカウント作成とログイン

#### 新規登録
1. アプリケーションを開き、「ログイン」ページにアクセスします
2. 「アカウントをお持ちでない方はこちら」リンクをクリックします
3. メールアドレスとパスワードを入力して「登録」ボタンをクリックします
4. または「Googleで登録」ボタンをクリックしてGoogleアカウントで登録します

#### ログイン
1. メールアドレスとパスワードを入力して「ログイン」ボタンをクリックします
2. または「Googleでログイン」ボタンをクリックします

### 2. Todoの作成

1. ログイン後、メイン画面の「新しいTodoを作成 (Cmd+K)」ボタンをクリックします
2. タイトルと説明を入力します（タイトルは必須、説明は任意）
3. 必要に応じてタグを選択します（複数選択可能）
4. 「作成」ボタンをクリックします

### 3. Todoの編集

1. Todo一覧から編集したいTodoの「編集」ボタンをクリックします
2. タイトル、説明、タグを変更します
3. 「更新」ボタンをクリックします

### 4. Todoの完了/未完了切り替え

1. Todo一覧から該当するTodoのチェックボックスをクリックします
2. 完了状態が即座に切り替わります

### 5. Todoの削除

1. Todo一覧から削除したいTodoの「削除」ボタンをクリックします
2. 確認ダイアログが表示されるので、「削除」ボタンをクリックします

### 6. タグの管理

#### タグの作成
1. ヘッダーの「タグ管理」リンクをクリックします
2. 「新しいタグを作成」ボタンをクリックします
3. タグ名を入力し、カラーを選択します
4. 「作成」ボタンをクリックします

#### タグの編集
1. タグ一覧から編集したいタグの「編集」ボタンをクリックします
2. 名前とカラーを変更します
3. 「更新」ボタンをクリックします

#### タグの削除
1. タグ一覧から削除したいタグの「削除」ボタンをクリックします
2. 確認ダイアログが表示されるので、「削除」ボタンをクリックします
   ※ タグが付いているTodoがある場合、自動的にそのTodoからタグが削除されます

### 7. フィルタリング

1. Todo一覧画面で、タグを選択してフィルタリングできます
2. 複数のタグを選択すると、OR条件で絞り込みされます
3. フィルターをクリアするには、すべてのタグの選択を解除します

## キーボードショートカット

- `Cmd+K` (Mac) / `Ctrl+K` (Windows/Linux): 新しいTodoを作成
- `Esc`: モーダルウィンドウを閉じる

## 技術仕様

### 対応ブラウザ
- Google Chrome (推奨)
- Mozilla Firefox
- Safari
- Microsoft Edge

### システム要件
- JavaScriptが有効なブラウザ
- インターネット接続（リアルタイム同期のため）

### データ保存
- すべてのデータはFirebase（Google Cloud Platform）に保存されます
- ユーザーのデータは完全にプライベートです

## トラブルシューティング

### ログインできない
- メールアドレスとパスワードが正しいか確認してください
- Googleアカウントでログインする場合、ブラウザのポップアップブロックが無効になっているか確認してください

### 変更が反映されない
- インターネット接続を確認してください
- ページをリロードしてみてください

### Todoが作成できない
- タイトルが入力されているか確認してください
- ネットワーク接続を確認してください

### タグが表示されない
- タグ管理ページでタグが作成されているか確認してください
- ページをリロードしてみてください

### アプリケーションが動作しない
- JavaScriptがブラウザで有効になっているか確認してください
- ブラウザのキャッシュをクリアしてみてください
- 別のブラウザで試してみてください

## プライバシーとセキュリティ

- ユーザーのデータはFirebaseのセキュリティルールによって保護されています
- 他のユーザーのデータにアクセスすることはできません
- パスワードは暗号化されて保存されます

## サポート

問題が発生した場合、以下の情報を開発者に提供してください：
- 使用しているブラウザとバージョン
- エラーメッセージ（表示されている場合）
- 問題が発生した操作の手順
